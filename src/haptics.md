## **API: Haptics**

### Your Task

Implement a screen where the user presses buttons that trigger different vibration/haptic patterns.
Possible ideas:

* A set of buttons: "Success", "Warning", "Error", "Light", "Medium", "Heavy"
* A fun “haptic piano” where each button vibrates differently

### Useful Imports

```ts
import * as Haptics from 'expo-haptics';
```

### Suggested API functions

* `Haptics.notificationAsync()`
* `Haptics.impactAsync()`
