## **API: Sensors**

### Your Task

Use one of Expo’s sensors (your choice).
Options:

* Accelerometer
* Gyroscope
* Pedometer
* Magnetometer

Possible ideas:

* Show live accelerometer values
* Rotate a square on screen based on device tilt
* Make a “shake to trigger” action

### Useful Imports

```ts
import { Accelerometer } from 'expo-sensors';
// or Gyroscope, Pedometer, Magnetometer
```

### Suggested API functions

* `Accelerometer.addListener()`
* `Accelerometer.setUpdateInterval()`
