## **API: Video**

### Your Task

Create a screen that plays a video.
Possible ideas:

* Auto-play a remote video
* Add buttons for play/pause/mute
* Add a progress slider (optional)

### Useful Imports

```ts
import { Video, ResizeMode } from 'expo-av';
```

### Suggested API usage

* `<Video source={{ uri: "https://..." }} />`
* `videoRef.current.playAsync()` (optional)
