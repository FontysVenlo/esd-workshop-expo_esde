# ESD template
## Instructions
First build the Docker image (inside `src` directory):
```bash
docker build -t expo-workshop:latest .
```

Then, run the container with `docker compose`:
```bash
docker compose -f compose.yaml up
```

> [!NOTE]
> Docker may require superuser privileges
